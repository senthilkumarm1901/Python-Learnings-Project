Pre-regression Steps - Standardization and Normalization¶

The process for getting data ready for a machine learning algorithm can be summarized in two main steps:

1. Data Pre-Processing - involving formatting (what if numerical variable is in string/object format), cleaning (missing value, outlier treatment), etc.,
2. Data Transformation/ Data Scaling

Many machine learning methods expect or are more effective if the data attributes have the same scale. Data rescaling is an important part of data preparation before applying machine learning algorithms

Two popular data scaling methods are

  1. normalization - refers to rescaling numeric attributes into the range 0 and 1
  2. standardization - refers to shifting the distribution of each attribute to have a mean of zero and a standard deviation of one (unit variance)

Important Points

  • Should/Can we normalize Categorical variables? 
      For regression and classification it would NOT matter if we perform scaling/normalization of categorical variables
  • Which normalization to use? 
      It is hard to know whether rescaling your data will improve the performance of your algorithms before you apply them. 
  It often can, but not always
  • There are other simple transformations also 
      df_normalized=df/df.max()
  or any custom transformation like log or exponential
