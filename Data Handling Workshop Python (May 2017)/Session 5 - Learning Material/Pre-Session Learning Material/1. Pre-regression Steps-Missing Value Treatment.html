Missing Value Treatment¶

Why Missing Values need to be treated?

  • A machine learning algorithm can fail when it contains missing values
  • Can give an incorrect view of the analysis if not treated

Two Main Steps:

1. Identification of Missing or Incorrect Values:
    - Missing values could be entirely missing in the data or have __0__ or a random character like __?__
2. Two Common Missing Value Treatment methods:
    1. Mean/ Mode/ Median Imputation: 
        * Generalized Imputation - apply across all missing values
        * Similar Case Imputation - treat different missing values differently
    2. K Nearest Neighbour Imputation (theory only):
        * the missing values of an attribute are imputed 
        using the given number of attributes that are most similar to the attribute whose values are missing
        * The similarity of two attributes is determined using a distance function
